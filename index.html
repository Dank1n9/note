多维数组
2026年05月22日
主要内容
多维数组： 二维数组： { 类似一个平面，相当于多个一维数组 元素含量= 行 × 列 行/列 下标从0开始 第一个[]意为行 第二个[]意为列 } 声明： 整数二维数组 { int a[X][X]; } 初始化二维数组 { int a[3][3]={1,2,3,4,5,6,7,8,9}; 当前a内容： [1 2 3] [4 5 6] [7 8 9] } 省略行长度 { int a[][X]; (X不能省略！！！) 【当行长度达到X时自动换行】 }
